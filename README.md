# 파이썬 학습 로드맵 (Python Learning Roadmap)

파이썬을 체계적으로 학습할 수 있는 단계별 로드맵과 학습 자료를 제공합니다.

## 📚 학습 단계

### 🌱 초급 (Beginner)
- 파이썬 기초 문법
- 변수, 데이터 타입, 연산자
- 제어문 (조건문, 반복문)
- 함수와 모듈

### 🌿 중급 (Intermediate) 
- 객체지향 프로그래밍
- 파일 처리와 예외 처리
- 라이브러리 활용
- 웹 크롤링 기초

### 🌳 고급 (Advanced)
- 웹 프레임워크 (Django, Flask)
- 데이터베이스 연동
- API 개발
- 배포와 운영
